# The Roots' Dental Clinic

Dental website template built with Vite, React, TypeScript, Tailwind CSS and shadcn/ui.

## Run locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deploy (Vercel)

- Framework: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
